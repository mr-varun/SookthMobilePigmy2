# Database Seeds

This directory contains seed data for testing and development.

## Usage

Place SQL seed files here to populate the database with test data:
- `test_agents.sql` - Sample agent data
- `test_customers.sql` - Sample customer data
- `test_collections.sql` - Sample collection data

## Note

Seed data is optional and used for development/testing purposes only.
Production data should be imported from the actual database backup.
